# AutoScroll

A Chrome extension for smooth, hands-free auto-scrolling — built for recording clean screen recordings without manual scroll interruptions.

## Features

- **Adjustable speed** — Slider from 10 to 500 pixels per second
- **Up or down** — Toggle scroll direction
- **Keyboard shortcuts** — Control scrolling without touching the mouse
  - `Alt+S` — Toggle scrolling on/off
  - `Alt+Up` — Increase speed
  - `Alt+Down` — Decrease speed
- **Invisible during recording** — All controls live in the extension popup, nothing appears on the page

## Install

1. Clone this repo or download as ZIP
2. Open Chrome → `chrome://extensions`
3. Enable **Developer mode** (top right)
4. Click **Load unpacked** → select the project folder

## Usage

1. Navigate to any webpage
2. Click the AutoScroll extension icon
3. Set your desired speed and direction
4. Click **Start**
5. Close the popup — scrolling continues
6. Use `Alt+S` to stop/start while recording
